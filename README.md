# NanoBioMat website

This repository contains the source files of the NanoBioMat academic website.

Website:
https://lenaiclartigue.github.io/NanoBioMat/

NanoBioMat is a research activity developed at CEISAM UMR 6230, Nantes Université, focused on magnetic nanoassemblies, biomaterials and nanomedicine.

## Website structure

The website is built as a static HTML/CSS website hosted with GitHub Pages.

Main pages:

* `index.html` — homepage
* `research.html` — research topics
* `projects.html` — funded projects
* `team.html` — team and supervision
* `collaborations.html` — national and international collaborations
* `facilities.html` — facilities and available techniques
* `publications.html` — selected publications and academic profiles
* `teaching.html` — teaching activities
* `news.html` — recent news
* `contact.html` — contact information

The visual style is defined in:

* `style.css`

Images, logos and illustrations are stored directly in the repository and used by the HTML pages.

## Maintenance

To update the website:

1. edit the relevant `.html` file;
2. update or add images if needed;
3. commit the changes to the `main` branch;
4. GitHub Pages automatically updates the website after a short delay.
